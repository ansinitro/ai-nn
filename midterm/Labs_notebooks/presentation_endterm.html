Final Exam: MNIST Recognition

Benchmarking Architectures

The AI Final Exam evaluates our capability to construct raw neural networks from the ground up to classify the MNIST handwritten digit database.


  • Model 1: Deep Neural Network (DNN)
    A fully connected Dense(128) -> Dense(64) topology requiring a flattened 1D input tensor.

  • Model 2: Convolutional Neural Network (CNN)
    A spatial Conv2D(32) -> MaxPooling2D() topology leveraging 2D feature mapping and pooling.

CNN vs DNN Final Conclusion

Both models trained for 5 epochs. The results perfectly demonstrate the inductive bias of Convolutional layers for computer vision tasks.

Architecture Train Accuracy Test Accuracy Parameters
Deep Neural Network (DNN) 0.9856 0.9768 109,386
Convolutional Network (CNN) 0.9922 0.9897 34,826

Conclusion: The CNN not only achieves higher testing accuracy but does so while requiring 3x fewer trainable parameters, definitively proving the superiority of spatial convolutions over linear dense mappings for image classification.
