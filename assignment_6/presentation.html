🐛 Bug Found & Fixed

The contrastive loss formula uses convention:

L = y·d² + (1−y)·max(M−d, 0)²
y=1 → similar (minimize d)  |  y=0 → dissimilar (push d > M)
❌ Before (wrong labels):
  • Normal-Normal pairs → y=0 → loss pushed them apart
  • Normal-Anomaly pairs → y=1 → loss pulled anomalies closer
  • Network trained backwards! Anomalies mapped near normals.
✅ After (correct labels):
  • Normal-Normal pairs → y=1 → normals cluster together
  • Normal-Anomaly pairs → y=0 → anomalies pushed away
  • kNN distance now correctly high for anomalous windows

Siamese Architecture

  • Two branches share one 1D-CNN embedding net (tied weights).
  • Each branch: (24×26) window → 128-dim embedding.
  • Lambda layer computes Euclidean distance between embeddings.
  • Trained with contrastive loss, margin M = 1.0.

Inference: Anomaly Scoring

Training Pairs

  • y=1 (similar) — Normal + Normal → minimize distance
  • y=0 (dissimilar) — Normal + Anomaly → push > margin
  • 60,000 train pairs / 15,000 val pairs

Scoring at Inference

  • Extract embeddings for 5,000 sampled normal train windows (reference library).
  • For each test window: compute kNN distance (k=1) to nearest reference.
  • High distance → anomaly, low distance → normal.

Threshold

  • Sweep over val-set kNN distances, pick threshold maximising F1.
  • Best threshold: 0.2655 (val F1 = 0.8803)

Contrastive loss decreases steadily. Train and val track closely — no overfitting with the corrected labels.

Conclusions

  • A label inversion bug in contrastive loss was identified and fixed — improving AUC by +2.6% and F1 by +3.0%.
  • The corrected Siamese 1D-CNN achieves AUC 0.9253 and F1 0.8437 for anomaly detection without explicit anomaly labels at training time.
  • kNN distance to reference normal embeddings is an effective, interpretable anomaly score.
  • Supervised classifiers (MLP, CNN, LSTM) remain superior when ground-truth labels are available.
  • Siamese + Autoencoder together offer complementary anomaly signals: AE detects global reconstruction failure; Siamese detects local distance from the normal manifold.
