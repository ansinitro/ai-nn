Anomaly Detection & Outlier Handling

Rigorous filtering is essential for stability in Neural Networks without compromising time-series continuity.

Strategic Procedure

  • Deployed Z-score methodology with a strict $4\sigma$ Threshold.
  • A 4$\sigma$ span ensures we trap true sensor failures (e.g., wind readings spiking to ungodly levels or extreme impossible drops) but retain realistic weather phenomenons.
Data Preservation Rule: Row deletion (`dropna`) is forbidden. Deletions destroy the perfect 10-minute step frequency mandated by sequence models (RNN/LSTM).
  • Outliers flagged with NaN.
  • Reconstructed via .interpolate(method='time') to respect the timestamp sequence elegantly.

Feature Engineering: Sequence Context

Pure raw states are suboptimal for sequence prediction. We mathematically embed historical context into each row so the model perceives momentum.

1. Historical Lags (Shift)

Given 10-minute intervals, 1 chronological hour evaluates to exactly 6 steps backwards.

Feature Name Meaning Implementation
lag_1 Temp 1 Hour prior shift(6)
lag_2 Temp 2 Hours prior shift(12)
lag_3 Temp 3 Hours prior shift(18)

2. Moving Averages (Rolling)

Filters instantaneous noise (like a sudden gust of wind cooling a sensor globally).

Feature Name Window Space Implementation
roll_30m 30 Minutes rolling(3).mean()
roll_60m 60 Minutes rolling(6).mean()
roll_120m 120 Minutes rolling(12).mean()

Note: The first 18 rows generated `NaN` due to missing history bounds. These 18 rows (representing 0.004% of total data) were safely dropped to prevent downstream ML poisoning.

Feature Engineering: Encoding & Normalization

Ensuring data formats map effortlessly into deep space learning spaces (TensorFlow/Keras architectures).

3. Categorical Structuring

Extracted `hour` integers were mapped to abstract states ('Morning', 'Day', 'Evening', 'Night').

  • Label encoding (0,1,2,3) was rejected because it implies false ordinal magnitude to Neural Networks.
  • One-Hot Encoding (OHE) was applied directly. `tod_Morning`, `tod_Night` acting as independent binary triggers.

4. Feature Normalization (Scikit-Learn)

Activation functions like Sigmoid / Tanh saturate and gradient descent slows dramatically on non-standard numeric widths.

  • Deployed Sklearn's MinMaxScaler() to crush all absolute metrics strictly inside [0, 1] boundaries.
  • (Future state: Fitted mapping must be exclusively fit on Train split to circumvent Data Leakage).
