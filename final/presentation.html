📊 Vector Database

FAISS Index Configuration

  • Type: IndexFlatIP
  • Metric: Inner Product (= cosine for normalized vectors)
  • Vectors: 1700
  • Dimensions: 768
  • Size: 5.0 MB

Why FAISS Flat?


  • Exact search (no approximation loss)
  • Fast for <10K vectors
  • CPU-based (saves GPU for LLM)
  • Sub-millisecond query time
Sanity Check: Query chunk[0] → score 1.000 ✅

Llama 2 7B Chat

PropertyValue
Quantization4-bit (3.8 GB model file GGUF)

Smart GPU Management: Auto-retry with fewer GPU layers if VRAM is insufficient (25 → 20 → 15 → 10 → CPU).
