Assignment 5
Denoising Autoencoder

Noise reduction and robustness analysis for the CNN classifier.

Problem Statement

Build a fully‑connected autoencoder that learns to reconstruct clean sensor signals from noisy inputs and evaluate how denoising improves the downstream 1D‑CNN classification on the Jena weather dataset.

Dataset

Same pre‑processed windows used in Assignment 4 (10‑minute intervals, 24‑step windows, target T (°C)).

Autoencoder Architecture

Symmetric dense network:
Input → 256 → 128 → 64 (bottleneck) → 128 → 256 → Input

Loss: Mean Squared Error, Optimizer: Adam (lr = 1e‑3). Trained for 50 epochs with early stopping.

Key Takeaways

  • Autoencoder successfully learns a compact latent representation.
  • Denoising recovers up to +2.6 % accuracy and +2.7 % F1 at high noise levels.
  • Robust preprocessing improves downstream model reliability.
  • Future work: explore convolutional autoencoders and variational models.
