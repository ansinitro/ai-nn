Architectures & Methodology

We implemented two distinct fully-connected neural network architectures to forecast the target variable T (degC) based on meteorological context features generated in Assignment 1.

1. Multi-Layer Perceptron (MLP)

  • 3 hidden layers: 64 ➔ 32 ➔ 16 neurons.
  • Activation function: ReLU (Rectified Linear Unit).

2. Deep Neural Network (DNN)

  • 5 hidden layers: 128 ➔ 64 ➔ 32 ➔ 16 ➔ 8 neurons.
  • Activation function: ReLU. Provides deeper representation capacity.

Training Parameters

  • Optimizer: Adam
  • Loss Function: MSE (Mean Squared Error)
  • Early Stopping: Monitor val_loss, Patience 5
  • Batch Size: 128 for 50 max epochs
